1) Le Volet des répértoires GitHub.

Le premier élément de ce Volet est un champ texte, vous pouvez y placer l'url d'un répertoire git (sous la forme "https://github.com/{user}/{nom_du_projet}").
Cliquer sur le bouton "+" pour ajouter le projet à la liste.
Le bouton "-" supprime le projet de la liste.
Dès qu'un projet est ajouté la Dashboard est mise à jour.

2) L'Identification

Pouvoir Ajouter, Suprimer et Visualisé les données d'un projet est bien beau, mais le souci c'est que toute modification sera perdue au prochain rafraîchissement de la page.
Pour pouvoir conserver la liste de vos projets une identification est possible.
Veuillez utiliser le compte "RomainR" avec comme mot de passe "IAGL" pour vous connecter.
Une fois que vous vous serez identifié, toutes les modifications faites à la liste des répertoires GitHub seront conservées et pourront être retrouvé à votre prochaine connexion.

3) La visualisation

Vous pouvez filtrer les éléments affichés en chiquant dessus dans la légende.
